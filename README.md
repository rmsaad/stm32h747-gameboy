# STM32H747 Gameboy Emulator

![Gameboy](images/gameboy.gif?raw=true "Title")

#### WIP 

A gameboy emulator for the STM32h747 Discovery Board. 

### Implemented 

- All Gameboy CPU instructions and timings  

- Interupt Handling 

- Joypad Support

- PPU Rendering 

- Memory Bank Controller 1
